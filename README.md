# HomeTool
该项目为传感器课程中的物联网课程设计，在此次课程设计中我主要是借助机智云平台以及ESP8266来实现的。
作品分为两部分，一部分为STM32的代码实现，另一部分是安卓APP的Java代码实现。此次上传的部分是STM32代码。
